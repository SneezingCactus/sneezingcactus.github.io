SneezingMod

A bonk.io client

by SneezingCactus

So, what is it?

SneezingMod is a client for Bonk.io that attempts to push the limits of the game and add game-changing features that will (hopefully) make for a more complete game experience.

This is not a hacked client, it's completely safe to use!

As of right now, SneezingMod isn't available to the public yet. But once it does, you'll be able to install it from the Chrome Web Store.
Firefox support isn't planned, but it will probably be available for it as well.

The biggest feature!

Ever wanted to make your own gamemodes? Well, today's your lucky day! SneezingMod includes a gamemode maker that lets you create your own gamemodes in a Scratch-like fashion. This, however, cannot be used on normal bonk.io rooms! You have to create and host a custom SneezingMod room. This is because your client has to send the gamemode data to the other clients in the room, and for that reason, it requires a custom server to function correctly.

Wait... custom servers?

Yes, custom servers! Custom servers not only let you use the gamemode maker, but it also has some perks, such as having more than 8 players in the room and having more than one host. It's also completely open-source, so you can modify it as much as you want. Instructions on how to make one yourself are available here!
